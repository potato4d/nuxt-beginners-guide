# プロキシを利用した API との連携

axios モジュールのプロキシ API を使ったプロキシ通信のサンプルです。実際の開発では、本番環境では `/api` へとアクセスを行った上で、 Cookie 認証を施したい場合などがあるかと思います。

そういった場合に、このサンプルが役立つこととなります。
